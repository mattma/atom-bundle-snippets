## Install

Go to `Atom > Preferences...` then search for **JavaScript Snippets** in Packages tab.

## Development

```sh
	$ cd ~/.atom/packages
	$ git clone https://github.com/mattma/atom-bundle-snippets.git
	$ cd atom-bundle-snippets
	$ apm install
	$ apm link
```
